Quoridor

The goal of the project was to develop a digital version of the board game Quoridor. I actively worked with a group of developers, mostly focusing on backend coding to ensure fluid gameplay and precise game mechanics. Git's version control system enabled us to work quickly and successfully integrate our code.

My cooperation, communication, and coordination skills were considerably improved as a result of working on this project in a group. It emphasized how crucial good communication is to getting a shared objective. A valuable skill I developed was using MySQL for effective database management, which allowed me to permanently store game data and player profiles.

This project's usefulness goes beyond the classroom. Quoridor's appeal to a wider audience can be improved by making the game available online. Its potential use is in giving players an online setting in which to play strategically, encouraging analytical thinking and problem-solving.
